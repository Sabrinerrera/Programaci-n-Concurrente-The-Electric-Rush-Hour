# Programaci-n-Concurrente-The-Electric-Rush-Hour
Simulación concurrente en Java del juego Rush Hour. El ID 0 debe salir por la columna 5. Los vehículos (hilos) consumen batería al moverse y se bloquean ante colisiones o falta de energía. Los cargadores (productores) recargan a los varados. Un monitor garantiza la exclusión mutua en el tablero de 6x6.
